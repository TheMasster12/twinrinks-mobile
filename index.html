Twin Rinks Mobile App

Mobile applications for Twin Rinks Adult Hockey.

The Android app is currently on the Google Play Store, but the iOS app is not near completion yet. Help in improving any application is greatly appreciated.

Please visit the Github page to fork the repository or check out the source code for either platform.
